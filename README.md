# Learn English Word

`learn-english-word`是一款在线背单词软件，感谢 [skywind3000](https://github.com/skywind3000) 提供的开源词库 [ECDICT](https://github.com/skywind3000/ECDICT)。

# HOW TO RUN

下载到用户根目录；

安装Python3，Flask，运行`app.sh`，在本地浏览器访问：

http://localhost:5000
